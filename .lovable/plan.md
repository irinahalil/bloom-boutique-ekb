

# План: Фото товаров в чате

## Обзор
AI-ассистент будет включать ссылки на фотографии товаров в формате markdown-изображений, а виджет чата будет их отображать.

## Изменения

### 1. Edge-функция `chat-assistant` — добавить image_url в каталог
- В запросе к products добавить поле `image_url`
- В каталоге для системного промпта добавить URL изображения к каждому товару
- В системном промпте указать AI: «Когда рекомендуешь товар, вставляй его фото в формате `![название](url)`»

### 2. `ChatWidget.tsx` — рендеринг изображений
- ReactMarkdown уже используется для рендеринга сообщений
- Нужно убедиться, что изображения рендерятся корректно (добавить кастомный компонент `img` в ReactMarkdown для стилизации: скруглённые углы, ограничение ширины, отступы)

## Файлы

**Изменить:**
- `supabase/functions/chat-assistant/index.ts` — добавить `image_url` в select и в системный промпт
- `src/components/ChatWidget.tsx` — добавить кастомный рендер изображений в ReactMarkdown
