

# План: Тест Telegram-уведомления

Пользователь нажал /start в боте. Нужно повторно вызвать edge-функцию `telegram-notify` для проверки доставки сообщения.

## Действие
- Вызвать `telegram-notify` с тестовым payload `new_chat` через `curl_edge_functions`
- Проверить результат — если `ok: true`, уведомления работают

## Технические детали
Один вызов edge-функции с тестовыми данными.
